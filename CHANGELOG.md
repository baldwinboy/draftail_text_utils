## [0.1.0] - 2026-05-23

### Added

- Text color feature with customizable color picker and predefined palette
- Highlight color feature with customizable color picker and predefined palette
- Font family feature with configurable font family presets
- Font size feature with configurable size range and presets
- Text alignment feature (left, center, right, justify) via block-level CSS class
- Color palette configuration via Django settings or external module
- Font family and URL loading from Django settings or external modules
- Toolbar control plugins and entity-based rich text features
- Wagtail admin asset injection (CSS, JS, inline data scripts)
- SVG icons for toolbar buttons
- Template tag for rendering font stylesheet links
